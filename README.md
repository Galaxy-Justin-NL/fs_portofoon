# fs_portofoon

📻 Fs-Radio - Geavanceerd Portofoon Systeem
Een intuïtief en geoptimaliseerd radiosysteem voor FiveM servers, volledig geïntegreerd met pma-voice en ox_lib. Dit script biedt een realistische manier voor burgers en hulpdiensten om over lange afstanden te communiceren via verschillende kanalen.

✨ Functionaliteiten
Gebruiksvriendelijke UI: Een strakke, moderne radio-interface die geopend wordt via een item.

Privékanalen: Beveiligde kanalen (1 t/m 10) die alleen toegankelijk zijn voor specifieke jobs (Politie, Ambulance, ANWB).

Publieke Kanalen: Onbeperkte kanalen voor burgers en criminelen (vanaf kanaal 11).

Animaties & Geluid: Inclusief radio-animatie (cop_radio) en realistische klik-geluiden bij het praten en wisselen van kanaal.

Volumebeheer: Pas je radio-volume direct in-game aan via de interface.

Optimale Performance: Draait op 0.00ms in rust en is extreem zuinig op de server resources.

🛠️ Vereisten (Dependencies)
Zorg dat je deze resources hebt geïnstalleerd voor een correcte werking:

es_extended (ESX Legacy)

pma-voice (Cruciaal voor de spraakverbinding)

ox_lib (Voor de menu's en notificaties)

ox_inventory (Voor het radio-item)
